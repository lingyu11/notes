# Essential C++

## Ch1 开始

1. 习惯上，程序执行无误时我们令main()返回0，若返回一个非零值，表示程序在执行过程中发生了错误
2. 两种初始化方法：

    | Initialize | Note |
    | ---- | ---- |
    | `int num_tries = 0;` |  |
    | `int num_tries(0);` | 构造函数语法，此法存在的原因是如果对象需要多个初值，<br> 这个方法就有用了，例如复数需要实部和虚部 |

3. 转义字符：

    | 转义字符 | 含义 |
    | ---- | ---- |
    | \n | 换行 |
    | \t | 制表符 |
    | \0 | null |
    | \' | ' |
    | \'' | '' |
    | `\\` | \ |
